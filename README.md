# ⚔️ RPG Maledetto

RPG a turni ispirato agli anime dark fantasy, sviluppato in Java con interfaccia grafica JavaFX.
Il giocatore interpreta uno stregone che esplora una mappa, combatte maledizioni,
raccoglie frammenti per sbloccare tecniche e affronta un boss finale.

---

## 🚀 Come eseguire il progetto

### Prerequisiti
- Java 25 (LTS)
- Gradle 9.1

### Istruzioni

```bash
git clone https://github.com/christiantaccar/RPG.git
cd RPG
```

### Build del progetto
```bash
./gradlew build
```

### Esecuzione
```bash
./gradlew run
```

---

## 🎮 Funzionalità principali

- Mappa esplorabile a griglia con fog of war
- Sistema di combattimento a turni con scelta delle tecniche
- Raccolta frammenti e sblocco manuale delle tecniche dall'inventario
- Sistema di salvataggio con 5 slot indipendenti
- Salvataggio rapido con tasto P
- Schermata Game Over con ripresa dall'ultimo salvataggio
- Schermata di vittoria al completamento

## 🤖 Uso di strumenti di AI

Durante lo sviluppo è stato utilizzato **Claude (Anthropic)** come strumento di supporto per:

- Supporto nella risoluzione di errori di compilazione e problemi JavaFX
- Suggerimenti sulla struttura delle classi e delle interfacce
  

L'AI è stata utilizzata come strumento di supporto allo sviluppo. Tutto il codice prodotto è stato
compreso, verificato e integrato personalmente. Le scelte architetturali, il design del gioco
e le decisioni implementative sono state prese dallo studente.

Per una descrizione dettagliata dell'uso dell'AI, consultare la **Wiki del repository**.
